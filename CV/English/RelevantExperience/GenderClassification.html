Employer: Uppsala University

Description of the job:

In my first year of master's studies I took a 5Hp course called Statistical Machine Learning at Uppsala University. The course was at master's level and was about learning the basic theory behind statical machine learning. At the end of the course we had a project about using Python to build a machine learning model that can classify the gender of leading actors (male/female) based on factors/features such as, gross income for the movie, gender of co-lead, etc.

Questions such as "Can Machine Learning methods be used in Classification problems?" and/or "How will the accuracy and presicion of the model be affected by the choice of features?" were answered.

Project:

The project was done in a group of 4 people. The code is stored at a private repository on GitHub.
