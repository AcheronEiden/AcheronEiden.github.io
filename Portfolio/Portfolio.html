Programming Projects

Object-Oriented Programming with Java

Nim (Private Repository)
A Java program that implements the game of Nim. The game is played between two players, where each player takes turns in picking out a specific number of sticks from a pile. The player who picks out the last stick loses the game. The program is part of the course "Object-Oriented Programming with Java" at Uppsala University. (pdf)
Geometric Figure Manipulation (Private Repository)
A Java program that manipulates geometric figures. The program uses the swing library to draw the figures (circle, rectangle, triangle). In addition to that, mouse clicks and movements are observed to interact with the figures (moves them, animate them or even add/remove figures). The program is part of the course "Object-Oriented Programming with Java" at Uppsala University. (pdf)
Solitaire (Private Repository)
A program that simulates a version of Solitaire called Napoleon's Grave. All automatic chosen moves are discarded and exchanged with human-chosen moves, giving the player much more freedom over the game. This project is part of the course "Object-Oriented Programming with Java" at Uppsala University. (pdf)
